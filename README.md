# DynamicTrees-PHC2
 A compatibility addon between Dynamic Trees and Pam's Harvest Craft 2 - Trees

# Missing features

- Maple treetap missing
- Maple, Paperbark and Cinnamon do not drop items when chopped  
- Ripe peppercorn is useless
